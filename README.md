## 💻 How To Install?
- Clone This Repo
```
git clone https://github.com/therevolt/ReactJS-FE-Tickitz
```
- Go To Folder Repo
```
cd ReactJS-FE-Tickitz
```
- Open Command/Terminal
- Type ``` npm start ``` To Start The Script
